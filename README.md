inumpy
======
Numpy extension for IPython.

Install
-------

```
pip install inumpy
```
or

```
pip install git+https://github.com/Lightslayer/inumpy  # development
```

The following only works for Jupyter < 5.0:

```
%install_ext https://raw.github.com/piti118/inumpy/master/inumpy.py
```

Usage
-----
```
%load_ext inumpy
```
